# Security Policy

## Supported Versions
The project is in active development. Latest `main` branch is supported.

## Reporting a Vulnerability
If you find a security issue:
1. Do NOT open a public issue with exploit details.
2. Email the repository owner or open a private security advisory on GitHub.
3. Provide steps to reproduce, impact assessment, and potential remediation ideas.

We will evaluate and respond within a reasonable timeframe. After a fix, details may be disclosed responsibly.

## Scope
- Authentication & authorization flows
- Data exposure (sensitive user fields, JWT handling)
- File upload handling (resume uploads / Cloudinary)

## Out of Scope (for now)
- Third-party service outages
- Browser extension interference

Thank you for helping keep Smart Hire-Hub secure.
